cmakeme_config¶

Find cmakeme. This module is called when using find_package(cmakeme). It brings in all of the cmakeme modules.

Result Variables¶

cmakeme_FOUND

True if cmakeme is found

CMAKEME_ROOT_DIR

Base directory of the cmakeme package
