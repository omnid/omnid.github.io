CMakeMe¶

A CMake package with utilities for installing executables and libraries, setting default compiler flags, accessing git commit hashes and hashes of source files at compile time, and generating documentation.

To access these modules, find_package(cmakeme). All the submodules will then be accessible.

  • cmakeme_pack (Create Installation Packages)
